# House Guide

A small, mobile-friendly static page with house-sitting instructions, hosted
free on GitHub Pages.

**Live:** https://philmet.github.io/house-guide/

## Editing

All the content lives directly in [`docs/index.html`](docs/index.html). Edit
the text there, commit, and push — GitHub Pages redeploys automatically.

- Photos are in [`docs/images/`](docs/images/).
- Styling is in [`docs/styles.css`](docs/styles.css).

> The page is public (anyone with the link can read it) and `noindex` so it
> won't show up in search engines. Keep alarm/keypad codes off the page — share
> those with sitters directly.

## Preview locally

```sh
npm run serve
```
